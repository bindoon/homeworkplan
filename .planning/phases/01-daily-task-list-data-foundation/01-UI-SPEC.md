# Phase 1 — UI Design Contract

> Visual and interaction contract for HomeworkPlan iOS MVP Phase 1. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

**Scope:** SETT-03, TASK-01 through TASK-07 — manual homework CRUD, today/all/settings tabs, subject management, SwiftData + iCloud. No import, AI parsing, recurring rules, or notifications in this phase.

**Sources:** `01-CONTEXT.md`, `REQUIREMENTS.md`, `docs/PRD.md` §3.6, `STACK.md`, `ARCHITECTURE.md`

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none (native SwiftUI) |
| Preset | not applicable — greenfield iOS app |
| Component library | SwiftUI system components (`TabView`, `NavigationStack`, `List`, `Section`, `Form`, `Sheet`, `DatePicker`, `Toggle`, `SwipeActions`) |
| Icon library | SF Symbols (navigation/actions) + subject emoji (科目标识) |
| Font | System font — `PingFang SC` (中文) / `SF Pro` (Latin) via `.font(.body)` etc.; no custom font files in Phase 1 |

**Platform defaults:** Follow Apple Human Interface Guidelines. Prefer semantic colors (`Color.primary`, `Color.secondary`, `Color.accentColor`, `Color(.systemGroupedBackground)`) over hardcoded hex where possible; hex values below are reference targets for custom theming if needed.

**shadcn gate:** Not applicable — React/web stack not used.

## Spacing Scale

Declared values in **points (pt)** — SwiftUI default unit; 1 pt ≈ 1 px @1x.

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4pt | Icon-to-label gap, inline badge padding |
| sm | 8pt | Row internal gaps, compact stack spacing |
| md | 16pt | List row horizontal inset, form field vertical spacing, section header bottom gap |
| lg | 24pt | Screen horizontal padding (non-list areas), sheet top padding |
| xl | 32pt | Empty state vertical spacing between illustration and copy |
| 2xl | 48pt | Major section breaks (empty state top inset) |
| 3xl | 64pt | Not used in Phase 1 |

Exceptions:
- **44pt minimum touch target** for all tappable controls (checkbox, tab bar items, toolbar buttons, FAB) per HIG
- **List row height:** system default (~44pt min); do not compress below HIG minimum
- **Tab bar:** system `TabView` height; do not custom-size

## Typography

| Role | Size | Weight | Line Height | SwiftUI mapping |
|------|------|--------|-------------|-----------------|
| Body | 17pt | 400 (regular) | 1.47 (~25pt) | `.font(.body)` — task content, form fields, settings rows |
| Label | 15pt | 400 (regular) | 1.33 (~20pt) | `.font(.subheadline)` — due date, section subtitles, metadata |
| Heading | 20pt | 600 (semibold) | 1.2 (~24pt) | `.font(.title3).fontWeight(.semibold)` — section headers (科目名), sheet titles |
| Display | 28pt | 600 (semibold) | 1.14 (~32pt) | `.font(.title2).fontWeight(.semibold)` — empty state heading only |

**Truncation:** Task content summary max **2 lines** with tail truncation (`.lineLimit(2)`). Subject name single line.

**Navigation title:** Large title on root tabs (`navigationBarTitleDisplayMode(.large)`); inline title in pushed sheets.

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#F2F2F7` / `Color(.systemGroupedBackground)` | App background, grouped list canvas, tab root surfaces |
| Secondary (30%) | `#FFFFFF` / `Color(.secondarySystemGroupedBackground)` | List rows, cards, sheet backgrounds, form sections |
| Accent (10%) | `#007AFF` / `Color.accentColor` | See reserved list below |
| Destructive | `#FF3B30` / `Color.red` | Swipe-delete action, destructive confirmation button |

Accent reserved for:
- **Primary CTA**「添加作业」button / toolbar button
- **Selected TabView tab** tint (今日 default selected)
- **Completion checkbox** when checked (`.tint(.accentColor)` on `Toggle`)
- **Today highlight** in date picker when viewing current date
- **「保存」**primary action in task edit/create sheet

Accent NOT used for: subject section headers, due-date labels, navigation chevrons, list row backgrounds, settings toggles (use system gray), or decorative emoji.

**Completed tasks:** `Color.secondary` strikethrough on content text; checkbox filled with accent. Completed tasks hidden from Today list (TASK-02) but visible in All tab.

**Due date urgency (optional semantic, not accent):** due today → `Color.orange` label; overdue → `Color.red` label; future → `Color.secondary`. Labels only, not row backgrounds.

## Screen Inventory

### Tab 1 — 今日 (TodayView)

| Region | Component | Behavior |
|--------|-----------|----------|
| Navigation bar | Large title「今日」+ trailing **添加作业** | CTA opens `TaskFormSheet` in create mode |
| Top | Compact `DatePicker` (.graphical or .compact per discretion) | TASK-06: switch viewing date; default today |
| Body | `List` grouped by `Subject` | TASK-02: incomplete tasks for selected date only |
| Section header | Emoji + subject name | One section per subject with ≥1 task; hide empty subjects |
| Row | Content (2 lines) + due date label + trailing `Toggle` checkbox | Tap row → edit sheet; toggle → TASK-03 complete + timestamp |
| Row swipe | Leading none; trailing **删除** (destructive) | TASK-05; confirm alert before delete |
| Empty | Centered illustration (SF Symbol `checklist`) + empty copy | When no incomplete tasks for date |
| Bottom | System `TabView` tab bar | Three tabs |

### Tab 2 — 全部 (AllTasksView)

| Region | Component | Behavior |
|--------|-----------|----------|
| Navigation bar | Large title「全部」 | |
| Body | `List` grouped by due date (section = date string) | Past + future tasks; includes completed |
| Section header | Relative date label e.g. **今天** / **昨天** / **6月20日 周五** | Sort descending by date |
| Row | Subject emoji + content + completion badge | Tap → edit; swipe delete same as Today |
| Completed row | Strikethrough content, checkmark icon | Visually distinct, not hidden |

### Tab 3 — 设置 (SettingsView)

| Region | Component | Behavior |
|--------|-----------|----------|
| Section | **科目管理** | TASK-07 |
| Rows | Default + custom subjects (emoji + name) | Swipe delete custom only; defaults non-deletable |
| Row action | **添加科目** | Sheet: name + emoji picker (preset emoji grid) |
| Section | **关于** | App name, version, brief description |

### Sheet — TaskFormSheet (create / edit)

| Field | Type | Validation |
|-------|------|------------|
| 科目 | Picker from `Subject` list + inline add shortcut | Required |
| 作业内容 | `TextField` multiline | Required, non-empty trimmed |
| 截止日期 | `DatePicker` date only | Required; default today for create |
| 备注 | `TextField` multiline | Optional |

Toolbar: **取消** (dismiss) · **保存** (primary, disabled until valid). TASK-01 create, TASK-04 edit.

## Interaction Patterns

| Pattern | Spec |
|---------|------|
| Default launch tab | **今日** (index 0) |
| Complete task | Toggle checkbox → immediate persist `isCompleted` + `completedAt`; row animates out of Today list |
| Edit task | Tap row anywhere except checkbox → present edit sheet with pre-filled values |
| Delete task | Swipe trailing → **删除** → confirmation alert → repository delete |
| Date browse (Today) | DatePicker changes query filter; title area may show **6月22日 周日** subtitle when not today |
| Subject on create | Picker defaults to last-used or first subject |
| iCloud sync | No dedicated UI in Phase 1 (SETT-03 backend); no sync status indicator unless error surfaces via save error copy |
| Accessibility | All interactive elements `accessibilityLabel` in Chinese; checkbox label =「标记 {content} 为完成」 |
