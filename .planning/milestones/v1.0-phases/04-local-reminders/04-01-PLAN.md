## Phase Goal

**As a** 家长, **I want to** 收到作业截止与重复任务的本地提醒, **so that** 不会忘记帮孩子完成作业。

<objective>
实现 ReminderService、NotificationBudgetManager、ReminderSettings、设置页提醒配置与权限引导，并联动 TaskRepository 与 RecurringTaskGenerator。
</objective>

## Artifacts this phase produces

- `ReminderSettings` — UserDefaults morning/afternoon times
- `ReminderNotificationID` — stable identifier helpers
- `ReminderNotificationBuilder` — builds UNNotificationRequest from task
- `NotificationBudgetManager` — 64 pending limit selection
- `ReminderService` — schedule/cancel/rescheduleAll + authorization
- `ReminderSettingsView` — settings UI for times and permission
- `TaskRepository.reminderService` — lifecycle hooks
- `RecurringTaskGenerator` + `ReminderService.schedule` after generation
